# 시티아트전자행정시스템
# City:Arts Online Administration System
------------------
### 시스템 소개
본 시스템은 시티아트의 각종 사무를 편리하게 수행하고자 만들어졌습니다.
본 시스템은 '시티아트 전자행정법'에 따라 운영됩니다.

### 할 수 있는 것
* 전체 공지사항
* 유저
  * 유저의 등록 및 삭제
  * 처벌의 집행 및 복권
  * OP 유저 목록 관리

* 자치단체
  * 자치단체 공동게시판
  * 개별 자치단체 게시판
  * 자치단체 신청 및 승인

* 철도
  * 철도노선 신청 및 허가
  * 도시철도 노선 등록

* 말머리 설정 기능
* 게시판 말머리별 모아보기 기능
* ACL Group을 통한 유저 관리체계
